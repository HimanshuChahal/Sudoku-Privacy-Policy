Sudoku Privacy Policy

Sudoku - The Brain Game is a mobile application developed to provide a suitable and the most effective brain game to its users. Sudoku was originated from Japan and is considered to be one of the most interesting games of all times. Please follow this link to learn more about the game - About Sudoku.

This is a Privacy Policy document which is focused on providing the information about the data usage of a user using the application. Following are the points that describes data usage by the app for the first version of release :-
- The application is only deployed on the Google Play Store and will only be available to Android users.
- The application does not require any exclusive permission from the user.
- There is no usage or transfer of the user's data to any external or internal source.
- The application does not require internet to provide the benefits of the game to the users.
